## NOTES

1. **Custom Classes**: The project uses custom component classes defined in `src/index.css` that layer on top of Tailwind utilities for consistency.

2. **Color System**: Uses CSS variables for all colors, allowing easy theming through root variables in light and dark modes.

3. **Responsive Design**: Heavy use of responsive prefixes (md:, lg:) with mobile-first approach.

4. **Design Language**: The class names and structure reflect a luxury furniture brand aesthetic with:
   - Serif fonts for headings (Playfair Display)
   - Sans-serif for body (Inter)
   - Warm, neutral color palette
   - Elegant spacing and typography

5. **Motion**: Uses Framer Motion for animations, complemented by Tailwind transition utilities.

6. **Dark Mode**: CSS variables support both light and dark themes with the `.dark` class.
